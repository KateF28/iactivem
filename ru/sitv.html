Система интерактивного тестирования и голосования

Комплект системы обеспечивает интерактивное голосование аудитории, позволяет получить различные варианты ответов на поставленные вопросы от большого количества людей и результаты голосования в процентах или количества на экран в графической и числовой форме.

Система голосования позволяет привлечь зрителей в процесс общения, отразить результаты дискуссии, получить докладчику обратную связь от аудитории, принять решение по отдельному вопросу, оценить результаты обучения и уровень знаний. Успешно используется на конференциях, съездах, в маркетинговых исследованиях

Пульт респондента

    • Размеры: 104мм x 49мм x15 мм

    • Количество кнопок: 7

    • Элемент питания: 2х AAA батарейки

    • Передача сигнала: > 100 метров

    • Угол передачи сигнала: 360 градусов

    • Срок службы элементов питания: ~ 2 лет при обычном использовании

  • 1. Окно передачи сигнала. Так как данная система радиочастотная, нет необходимости направлять пульт в сторону ресивера

  • 2. и 3. Индикаторы. При нажатии на кнопки пульта индикатор 3 загорается красным цветом, подтверждая работоспособность пульта. Индикатор 2 загорается зеленым цветом (при подключенном приемнике), подтверждая принятие сигнала от пульта ресивером. Прием сигнала ресивером занимает меньше секунды

  • 4. ID пульта. Пульт каждого ученика имеет уникальный номер (ID ученика), который может быть легко переустановлен преподавателем на ПК. Если пульт ученика утерян или поврежден, его можно заменить новым пультом, присвоив ему бывший ID ученика

  • 5. «Поднятие руки» - данная кнопка используется в некоторых режимах опросов (тест на время, поднятие руки) для определения респондента, который имеет право первым отвечать на вопрос, который ставится

  • 6. Кнопки выбора варианта ответа. Данная система предусматривает только один правильный ответ. Максимальное количество вариантов ответов - 6

Пульт лектора

    • Размеры: 104мм x 49мм x 15мм

    • Количество кнопок: 13

    • Элемент питания: 2х AAA батарейки

    • Передача сигнала: > 100 метров

    • Угол передачи сигнала: 360 градусов

    • Срок службы элементов питания: 1 год при обычном использовании

  • 1. Окно передачи сигнала

  • 2. и 4. Лазерная указка

  • 3. Индикатор. Загорается при нажатии функциональных кнопок

  • 5. «Запуск / Выход». Используется для запуска/закрытия презентации MS PowerPoint, выхода из режима тестирования для сохранения результатов

  • 6. «Отчет». Используется для вывода на экран суммарного количества баллов по каждому респонденту на данный момент тестирования. В данном списке студенты ранжируются по количеству набранных баллов. Первым отображается студент, набравший наибольшее количество баллов и ответивший быстрее всех (при одинаковом количестве баллов учитывается время, затраченное на ответ)

  • 7. «Результат». По истечении времени, отведенного на конкретный вопрос, нажатие на эту кнопку отобразит диаграмму с общим распределением ответов по вариантам ответов

  • 8. "Переключатель режимов". Переключение режимов опроса во время презентации в Multi режиме

  • 9. «?». Используется для мгновенной обратной связи в Multi режиме. Используется, например, с устным вопросом: «Вам всем понятно?»

  • 10. «К предыдущему слайду»

  • 11. «OK». Кнопка подтверждения

  • 12. «Стоп». Используется для остановки сбора ответов респондентов

  • 13. «Старт» и «Пауза». Используется для запуска сбора ответов респондентов, при повторном нажатии приостанавливает таймер, в то время как ресивер продолжает принимать ответы респондентов

  • 14. «F2» и 16 «F1». Вспомогательные функциональные кнопки

  • 15. «Переход к следующему слайду»
